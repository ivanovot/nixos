# NixOS Configuration

Персональная конфигурация NixOS с декларативным управлением системой и приложениями.

## Установка
```bash
git clone git@git.domow.su:timo/nixos.git
```
или
```bash
git clone git@github.com:ivanovot/nixos.git
```

```bash
./nixos/src/rebuild.sh
```

# Установка конфигов
```bash
./nixos/src/congig.sh
```
